# 🧪 Laboratório de Estudos - Python & Dados

Este é o meu espaço pessoal de experimentação e aprendizado autodidata. Aqui eu compartilho códigos, resoluções de desafios e pequenos projetos que desenvolvo em casa assistindo a tutoriais, lendo documentações e estudando por conta própria.

## 🎯 Objetivo
Aprofundar meus conhecimentos nos fundamentos de programação e dar os meus primeiros passos em direção à área de Engenharia de Dados.

## 🧠 O que você vai encontrar aqui?
* `automacoes/` - Scripts de automação simples com Python.
* `manipulacao-dados/` - Testes de leitura e limpeza de arquivos (TXT, CSV).
* `exercicios-logica/` - Desafios de lógica de programação.
* `sql-praticas/` - Consultas e testes com Bancos de Dados.

## 🚀 Tecnologias Utilizadas
